Acá van empezando a vivir las rutinas descargables de Omega. Podés bajar una rutina sola para usar en Cargar rutina o bajar el pack completo para sumarlas a tu base personal.
Entrar a Omega
Cómo se usa

Rutina individual: descargás el JSON, entrás al bloque correcto en Omega y usás Cargar rutina.

Pack completo: descargás el JSON grande y lo podés cargar como base personal al entrar a la app.

Esto es una primera base. Después podemos agregar autores, tags, nivel, equipamiento y una curaduría mucho más grande.
